# MLflow Tracking Skill

## Quick Start

This skill provides patterns and templates for effective MLflow experiment tracking on Databricks.

## Contents

- `SKILL.md`: Comprehensive guide to MLflow tracking patterns
- `templates/`: Reusable code templates
- `examples/`: Working examples for common scenarios

## Usage

1. Review `SKILL.md` for patterns and best practices
2. Use templates as starting points for your experiments
3. Reference examples for specific frameworks

## Key Files

- `templates/experiment_tracking_template.py`: Basic experiment tracking
- `examples/sklearn_classification.py`: Complete sklearn example
- `examples/hyperparameter_tuning.py`: Hyperopt integration

## Prerequisites

- Databricks ML runtime
- MLflow installed
- Unity Catalog access (for model registry)
