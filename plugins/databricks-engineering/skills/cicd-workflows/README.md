# CI/CD Workflows Skill

GitHub Actions and CI/CD patterns for Databricks.

## Templates

- `.github/workflows/ci.yml` - Continuous Integration
- `.github/workflows/deploy.yml` - Deployment workflow
- `test_runner.sh` - Test execution script
- `deployment_script.py` - Deployment utilities

## Examples

- `full_cicd_pipeline/` - Complete CI/CD setup
- `blue_green_deployment/` - Blue-green deployment

## Usage

1. Configure GitHub Actions workflows
2. Set up Databricks CLI authentication
3. Implement testing and validation
4. Automate deployment to environments
