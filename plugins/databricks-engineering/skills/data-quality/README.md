# Data Quality Skill

Comprehensive data quality patterns using Great Expectations, DLT expectations, and custom validators.

## Templates

- `great_expectations_suite.py` - GX suite builder
- `custom_validators.py` - Reusable validation functions
- `dlt_quality_checks.py` - DLT pipeline quality checks
- `data_profiling.py` - Data profiling utilities

## Examples

- `customer_data_validation.py` - Complete validation example
- `pipeline_quality_monitoring.py` - Quality monitoring setup

## Usage

1. Choose appropriate quality framework (GX or DLT)
2. Define quality rules based on data dimensions
3. Implement validation in pipeline
4. Monitor quality metrics over time
