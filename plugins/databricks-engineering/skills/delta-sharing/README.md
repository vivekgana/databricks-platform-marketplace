# Delta Sharing Skill

Delta Sharing setup, monitoring, and consumer management.

## Templates

- `share_configuration.py` - Share setup
- `recipient_management.py` - Recipient management
- `usage_monitoring.py` - Usage tracking

## Examples

- `external_data_sharing.py` - External partner setup
- `multi_tenant_sharing.py` - Multi-tenant sharing

## Usage

1. Create shares with appropriate tables
2. Set up recipients with authentication
3. Grant access permissions
4. Monitor usage and access patterns
