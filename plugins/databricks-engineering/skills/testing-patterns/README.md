# Testing Patterns Skill

pytest fixtures and integration testing patterns for Spark applications.

## Templates

- `conftest.py` - Pytest configuration
- `spark_fixtures.py` - Reusable Spark fixtures
- `dataframe_assertions.py` - DataFrame test utilities
- `mock_data_generators.py` - Test data generation

## Examples

- `test_transformation_pipeline.py` - Unit test example
- `test_integration_workflow.py` - Integration test example

## Usage

1. Set up pytest with Spark fixtures
2. Create test data with fixtures
3. Write unit tests for transformations
4. Add integration tests for end-to-end workflows
