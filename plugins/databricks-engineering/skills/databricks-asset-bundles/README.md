# Databricks Asset Bundles Skill

Modern deployment with DAB and multi-environment configurations.

## Templates

- `databricks.yml` - Main bundle configuration
- `environments/dev.yml` - Development environment
- `environments/prod.yml` - Production environment
- `deployment_workflow.py` - Deployment utilities

## Examples

- `complete_bundle_project/` - Full bundle structure
- `multi_workspace_deployment/` - Multi-workspace setup

## Usage

1. Define resources in databricks.yml
2. Configure environment-specific settings
3. Validate bundle configuration
4. Deploy to target environments
